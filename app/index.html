Let’s get started!

Setting up your project is as easy as 1—2—3!

To setup your project, cd into your project directory and do the following...

  1. Install bower, grunt-cli and compass modules globally. This is a one time task.

    npm install -g bower
npm install -g grunt-cli
gem install compass

  2. Set up local node dependencies. This installs all the node dependencies into the node_modules directory.

    npm install

  3. Set up the UI component dependencies. This installs all the UI dependencies into the app/components directory.

    bower install

… and that’s all! You’re good to go!
